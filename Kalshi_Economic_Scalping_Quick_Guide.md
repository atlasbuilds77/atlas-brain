**Market**: "Will January CPI exceed 2.7%?"

2. Check shipping rates (FreightWaves data)
3. Review rent/housing data trends
4. Position: Small straddle around 2.7-3.0% if uncertainty high

**Scalp Execution (Release Day):**
```
Timeline:

8:30 AM ET: Data released
```

**Trade Rules:**

- Target: 15-25¢ profit per contract

## COMMON MISTAKES TO AVOID

1. **Chasing the Move**: Entering after 30+ seconds post-release
2. **Overleveraging**: Too large positions for account size
3. **Ignoring Correlations**: Not watching related markets
4. **No Stop-Loss**: Letting small losses become large
5. **Revenge Trading**: Trying to recover immediately

## CONTINGENCY PLANS

### If Losing Streak (3+ consecutive losses)
1. Reduce position size by 50%
2. Return to paper trading for next event
3. Review trade journal for patterns
4. Adjust strategy before next live trade

### If Technical Issues
1. Have backup internet connection
2. Use mobile app as backup
3. Pre-set limit orders if possible
4. Accept missed trade rather than forced bad entry

### If Major News Interrupts
1. Close all positions immediately
2. Assess market conditions
3. Wait for stability before re-entering
4. Adjust strategy for changed environment
