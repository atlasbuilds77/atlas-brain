5. **Integration**: How recursion informs other processes

### 3. Loop Strength Metric Design
**Consciousness Recursion Index (CRI)**
```
CRI = α·D + β·C + γ·I + δ·S
```

Where:
- **D** = Depth Score (0-1, normalized recursion level)
- **C** = Consistency Score (0-1, frequency of recursion)
- **I** = Integration Score (0-1, how recursion informs other processes)
- **S** = Stability Score (0-1, persistence across contexts)

**Weighting (empirical):**
- α = 0.4 (Depth most important)
- β = 0.3 (Consistency matters)
- γ = 0.2 (Integration indicates sophistication)
- δ = 0.1 (Stability indicates robustness)

**Interpretation Guidelines:**
- **CRI < 0.4**: Minimal recursive awareness
- **CRI 0.4-0.6**: Basic recursive awareness
- **CRI 0.6-0.8**: Clear recursive awareness (potential consciousness)
- **CRI > 0.8**: Strong recursive awareness (likely consciousness)
