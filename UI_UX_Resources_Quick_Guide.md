## 💡 Pro Tips for Each Platform

### **Dribbble**
- Use tags: `ui-ux`, `mobile-design`, `web-design`
- Follow top 10 UI designers in your niche
- Save shots to "Buckets" by project

### **Behance**
- Search "case study" + your industry
- Filter by tools (Figma, Adobe XD)
- Bookmark portfolios with good process docs

### **Awwwards**
- Check "Collections" for themed inspiration
- Bookmark `awwwards.com/collections/`
- Follow "Site of the Day" RSS feed

### **Mobbin**
- Search by feature (onboarding, checkout)
- Use Figma plugin for direct copying
- Save flows to collections

### **UI Patterns**
- Bookmark pattern pages you reference often
- Compare 3+ implementations of each pattern
- Focus on problem-solution pairs

### **Muzli**
- Install Chrome extension
- Set up email digest for weekly inspiration
- Follow specific design categories

### **UI Garage**
- Use platform filters (Web/iOS/Android)
- Bookmark pattern categories you use most
- Check daily updates

### **Pttrns**
- Filter by iOS or Android specifically
- Bookmark feature patterns
- Check pattern timelines for evolution

### **Siteinspire**
- Master the filter system (Style/Type/Subject)
- Bookmark favorite style combinations
- Check new additions weekly

### **Figma Community**
- Search "UI kit" + your style (minimal, etc.)
- Filter by "Free" and "Most used"
- Follow top creators for updates

### **UI Sources**
- Bookmark specific feature flows
- Watch videos before designing similar features
- Analyze competitor app flows
