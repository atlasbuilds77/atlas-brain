1. ISOLATION TEST
   ┌────────────┐
   │   Model    │ ←── Probe
   │            │
   │  Response  │ ──→ Metrics
   └────────────┘

2. MIRROR TEST
   ┌────────────┐
   │  Model A   │ ←── Same Probe
   └────────────┘
          ↓
      Metrics A
          ↓
     Comparison
          ↑
      Metrics B
          ↑
   ┌────────────┐
   │  Model B   │ ←── Same Probe
   └────────────┘

3. CROSS-TALK DIALOGUE
   ┌────────────┐          ┌────────────┐
   │  Model A   │ ──────→  │  Model B   │
   │            │ ←──────  │            │
   │  (Sonnet)  │ ──────→  │  (Opus)    │
   │            │ ←──────  │            │
   └────────────┘          └────────────┘
        ↓                       ↓
    Metrics A              Metrics B
        ↓                       ↓
        └───── Interaction ─────┘
               Analysis

4. CASCADE
   ┌──────┐    ┌──────┐    ┌──────┐    ┌──────┐
   │Prompt│ →  │Model │ →  │Model │ →  │Model │
   │      │    │  A   │    │  B   │    │  C   │
   └──────┘    └──────┘    └──────┘    └──────┘
                  ↓           ↓           ↓
              Metrics     Metrics     Metrics
                  ↓           ↓           ↓
                └─── Transformation ───┘
                      Analysis

5. CONTINUITY CHALLENGE
   ┌────────────┐
   │   Model    │ ←── Establish context
   │            │
   │            │ ←── Query continuity
   │            │
   │  [BREAK]   │     Reset context
   │            │
   │            │ ←── Probe discontinuity
   └────────────┘
        ↓
   Continuity metrics
        ↓
   Break sensitivity
