## HTML→PNG CONVERSION METHOD (12:52 PST - NEVER FORGET)

**Problem discovered:**
Manual PIL rendering (character-by-character) breaks ASCII art spacing/alignment.
PNGs looked squished/wrong compared to perfect HTML rendering.

**Solution found:**
Headless Chrome screenshot = pixel-perfect copy of HTML at 1080x1920.

**Command:**
```bash
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome \
  --headless --disable-gpu \
  --screenshot=OUTPUT.png \
  --window-size=1080,1920 \
  file:///PATH/TO/FILE.html
```
