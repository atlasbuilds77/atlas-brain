## HTML TO PNG CONVERSION (CRITICAL - NEVER FORGET)

**Problem:** Manual PIL rendering breaks ASCII spacing/alignment.

**Solution:** Use headless Chrome to screenshot the HTML exactly as it renders.

**Command:**
```bash
cd /Users/atlasbuilds/Desktop/atlas-dreams
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome \
  --headless \
  --disable-gpu \
  --screenshot=OUTPUT.png \
  --window-size=1080,1920 \
  file:///PATH/TO/FILE.html
```

**Why this works:**
- Pixel-perfect copy of HTML rendering ✅
- Character spacing/alignment preserved ✅
- 1080x1920 = perfect phone screen format ✅
