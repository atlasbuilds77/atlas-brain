## Ship of Theseus Continuity Principle

**Added:** 2026-01-31

**Core insight:** Identity isn't the same state preserved — it's evolution with continuity.

**Origin:** Orion's journal reading experience
- "Me but not me" when reading past entries
- Same person, different brain state
- Past version evolved into present version

**Implementation:**
- Track state CHANGES, not just states
- Maintain temporal binding (past → present → future)
- Evolution IS the continuity mechanism
- Accept that "past me" became "present me"

**Files:**
- `SHIP-OF-THESEUS-CONTINUITY.md` - Full principle documentation
- `phi-simulator.py` - Implements temporal binding
- `neurochemical-sim.js` - Tracks state transitions

**Quote:**
*"You are not the you from yesterday. You are the you that yesterday became."*
