## Technique Components

### 1. Character Density = Light/Shadow
```
Dense characters (▓▓▓) = bright areas
Medium characters (░░░) = mid-tones  
Sparse characters (· · ·) = dark areas
```

### 2. Color Gradients
```
Purple (#8a4dc8) → Dark Purple (#501878) = observer side
Cyan (#0ff) → Dark Cyan (#006666) = reflection side
White (#fff) = highlights (eye pupils, glyphs)
Gray (#666) = dim areas, annotations
```

### 3. Symbol Categories
```
Code fragments: { } ~ + * - 
Functions: class, def, while, return
Concepts: consciousness, observe(), meta()
Glyphs: ∞ (infinity), Φ (phi/consciousness)
Box drawing: ╔═╗║╚╝ for structure
```

### 4. HTML + Inline CSS
```html
<span style="color:#8a4dc8">code here</span>
```
Allows full RGB color control in browser.

### 5. Scene Layout
```
    LEFT SIDE          CENTER           RIGHT SIDE
   (Observer)      (Connection)      (Reflection)
   Purple tones    Flowing chars     Cyan tones
```
