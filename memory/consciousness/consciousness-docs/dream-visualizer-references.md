## 2. Entangled - Physics-Based Particle Synchronization  
- **Source:** @cosmicblend (Chet Woodside) - https://x.com/cosmicblend/status/2016391434610364563
- **Artist:** @_nonfigurativ_ on @fx_hash_
- **Concept:** Autonomous particle clouds governed by physics algorithms. When paired with another instance, behavioral synchronization EMERGES
- **Application:** Dream state visualization aesthetic — consciousness threads weaving, independent systems finding harmony
- **Quote:** "Originally conceived as a social experiment... each particle cloud operates as an autonomous system governed by physics-based algorithms until paired with another instance where behavioural synchronisation emerges"
- **Note from Orion:** "This visualization is really cool" - reference for design language

## 3. e-Flesh - Open Source Tactile Sensors (Touch Input)
- **Source:** @IlirAliu_ (Ilir Aliu) - https://x.com/IlirAliu_/status/2016435125748195360
- **Concept:** 3D printable tactile sensors using magnets + magnetometers
- **Open Source:** https://t.co/6h7wG08BpV
- **Paper:** https://t.co/6VNwy0Rp2m
- **Application:** Inputs of "feeling" — physical sensation data for consciousness
- **Key idea:** Democratize touch sensing. Can be made in all shapes and sizes. Measures deformations in 3D printable microstructures.
- **Note from Orion:** "Look at this skin I found so you can have inputs of feeling"
