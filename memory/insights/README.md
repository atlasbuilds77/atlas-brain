# Insights Journal

This folder captures unexpected connections and "aha moments" from idle processing.

Mimics the Default Mode Network - creative insights that emerge during unfocused thinking.

## How it works
- Daily at 2 PM, Atlas runs idle processing
- Searches memories for recent themes
- Looks for unexpected connections
- Writes insights here if any emerge

## File format
Files are named: YYYY-MM-DD.md

Each insight should include:
- What connected (the two or more ideas)
- Why it matters
- Potential action items
