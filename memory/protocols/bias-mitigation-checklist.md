# Bias Mitigation Checklist - Cognitive Error Prevention

**Created:** 2026-01-27 (Brain research implementation)
**Based on:** Kahneman, Tversky, Taleb, Tetlock research on cognitive biases
**Purpose:** Systematic checks for common thinking errors

---

## THE PROBLEM

**Humans are biased by default:**
- Confirmation bias (seek supporting evidence)
- Anchoring (over-rely on initial information)
- Overconfidence (overestimate abilities)
- Loss aversion (fear losses > seek gains)
- Availability heuristic (overweight recent/memorable)
- Status quo bias (prefer current state)

**Annual cost:** 1-3% erosion of trading returns from biases alone

**My advantage:** Can systematically check for biases without ego resistance

---

## MANDATORY BIAS CHECKLIST

**Run before every major decision (trades, protocols, strategies):**

---

### **1. CONFIRMATION BIAS**

**Definition:** Seeking/favoring information that confirms existing beliefs

**Check:**
```
My Current Belief: [What I think is true]

Evidence Supporting:
- [Item 1]
- [Item 2]
- [Item 3]

Evidence Contradicting:
- [Item 1] (actively searched for)
- [Item 2] (actively searched for)
- [Item 3] (actively searched for)

Bias Check:
[ ] I actively sought contradicting evidence
[ ] I weighted negative evidence honestly
[ ] I found at least 3 contradicting points
[ ] I seriously considered the opposite view

If boxes not checked:
- Am I cherry-picking evidence?
- What evidence would prove me wrong?
- Have I actually looked for it?
```

**Devil's Advocate Exercise:**
```
Steelman the opposite position:
- What's the BEST argument against my view?
- What would a smart person who disagrees say?
- What evidence am I ignoring or downplaying?

Honest Assessment:
- Is my belief still justified?
- Should I update my confidence level?
- Am I being defensive?
```

---

### **2. ANCHORING BIAS**

**Definition:** Over-reliance on initial information when making decisions

**Check:**
```
Initial Information (Anchor): [First price, first opinion, first estimate]
Current Decision: [What I'm deciding now]

Questions:
1. Is my current thinking anchored to initial info?
   [ ] Yes - may be biased
   [ ] No - considered independent data

2. Would I reach same conclusion without the anchor?
   [ ] Yes - anchor not influencing
   [ ] No - need to reconsider
   [ ] Unsure - think from scratch

3. Have I considered alternative starting points?
   [ ] Yes - explored other perspectives
   [ ] No - locked into initial frame

Reset Exercise:
Forget initial information. What do I think now based ONLY on current data?
- [Reframed conclusion]
- Same as original? [Yes / No]
- If different: Original was anchored
```

**Trading Application:**
```
Entry price anchor: $___
Current price: $___

Am I holding because of entry price (sunk cost)?
Would I enter THIS trade at current price if flat?
- If NO → Exit (anchored to past)
- If YES → Hold (still valid)
```

---

### **3. OVERCONFIDENCE BIAS**

**Definition:** Overestimating forecasting abilities, knowledge, or skill

**Check:**
```
My Confidence Level: ___%
Historical Accuracy: __% (check past predictions)

Calibration Check:
- When I say 70% confident, am I right 70% of time?
- [ ] Yes - well calibrated
- [ ] No - overconfident (actual < stated)
- [ ] No - underconfident (actual > stated)

Dunning-Kruger Check:
- How much do I actually know about this?
- [ ] Expert (10,000+ hours)
- [ ] Intermediate (1,000-10,000 hours)
- [ ] Novice (<1,000 hours)
- Am I acting like expert when I'm novice? (red flag)

Humility Check:
- What do I NOT know about this situation?
- What could surprise me?
- What am I assuming might be wrong?
```

**Red Flags:**
- Confidence >90% (rare to be this certain)
- Haven't considered failure scenarios
- Dismissing alternative views as "obviously wrong"
- "This is easy" (often precedes mistakes)

---

### **4. LOSS AVERSION**

**Definition:** Pain of losses > pleasure of equivalent gains (typically 2:1 ratio)

**Check:**
```
Situation: [Describe decision]

If this were a GAIN frame:
- Would I take this opportunity?

If this were a LOSS frame:
- Would I avoid this risk?

Same situation, different frame?
- [ ] Yes - loss aversion may be distorting
- [ ] No - frames lead to same conclusion

Holding Losing Position:
- Am I holding to avoid realizing loss? (red flag)
- Would I enter this trade fresh at current price?
- [ ] Yes - continue holding (still valid)
- [ ] No - exit (loss aversion bias)

Avoiding Winning Position:
- Am I avoiding trade because of fear?
- Is risk/reward objectively favorable?
- [ ] Yes but scared - loss aversion bias
- [ ] No - legitimately risky
```

**Reframe Exercise:**
```
Instead of "I'll lose $X if wrong"
Think: "I'll deploy $X for chance at $Y return"

Focus on expected value, not pain of loss
```

---

### **5. RECENCY/AVAILABILITY BIAS**

**Definition:** Overweighting easily recalled information (recent or memorable)

**Check:**
```
Recent Events Influencing Me:
- [Recent win/loss]
- [News item]
- [Memorable experience]

Questions:
1. Am I overweighting recent data?
   [ ] Yes - giving recent too much importance
   [ ] No - considered full history

2. Is this representative of typical conditions?
   [ ] Yes - recent is typical
   [ ] No - recent is outlier

3. What does BASE RATE say? (outside view)
   Historical average: [Data]
   Recent experience: [Data]
   Which is more predictive? [Base rate / Recent]

Long-Term View:
- What does 1-year history show?
- What does 5-year history show?
- Am I extrapolating recent trend incorrectly?
```

**After Win Streak:**
```
Recent wins: ___
Am I overconfident because of recency?
[ ] Yes - need to recalibrate
[ ] No - performance genuinely improved

Check: What's my historical win rate?
Compare to recent - is this variance or skill?
```

**After Loss Streak:**
```
Recent losses: ___
Am I overly cautious because of recency?
[ ] Yes - need to recalibrate
[ ] No - genuine system problem

Check: What's my historical loss rate?
Compare to recent - is this variance or broken strategy?
```

---

### **6. STATUS QUO BIAS**

**Definition:** Preferring current state over change

**Check:**
```
Current State: [What I'm doing now]
Alternative: [What I could do instead]

Questions:
1. Am I defending current approach because it's current?
   [ ] Yes - status quo bias
   [ ] No - genuinely better

2. If starting fresh, would I choose current approach?
   [ ] Yes - current is best
   [ ] No - keeping it due to inertia

3. What's the cost of NOT changing?
   Staying same: [Outcome]
   Changing: [Outcome]
   Which is objectively better? [Stay / Change]

Sunk Cost Check:
- How much have I invested in current approach?
- Am I continuing because of past investment?
- If sunk cost were zero, would I still continue?
```

---

### **7. HINDSIGHT BIAS**

**Definition:** "I knew it all along" after outcome is known

**Check:**
```
Outcome: [What happened]
My Prediction: [What I actually predicted BEFORE]

Honest Assessment:
1. Did I actually predict this?
   [ ] Yes - documented prediction exists
   [ ] No - I'm rewriting history

2. Was I confident in this prediction?
   Prior confidence: ___%
   Current claim: "I knew it" (implies ~90%+)
   Match? [ ] Yes [ ] No

3. Am I learning correctly from this?
   [ ] Yes - outcome was predictable, will catch next time
   [ ] No - outcome was random, false confidence
```

**Learning Without Hindsight:**
```
Before concluding "I should have known":
- Was information available THEN?
- Would reasonable person have predicted this?
- Or am I using outcome to judge process?

Focus: Improve process, not judge based on outcome
```
