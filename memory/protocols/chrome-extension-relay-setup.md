# Chrome Extension Relay - Quick Setup (When You Get Home)

## What This Fixes
Lets me control YOUR existing Chrome tabs instead of opening a separate browser. Faster, cleaner, no more switching.

## 3-Minute Setup (Do This When Home)

### Step 1: Open Chrome Extensions
```
1. Open Chrome
2. Go to: chrome://extensions
3. Toggle ON "Developer mode" (top right)
```

### Step 2: Load Extension
```
4. Click "Load unpacked"
5. Paste this path: ~/.clawdbot/browser/chrome-extension
   (Or navigate to it)
6. Extension loads with Clawdbot logo
```

### Step 3: Pin & Attach
```
7. Click puzzle piece icon (top right)
8. Pin "Clawdbot Browser Relay" 
9. Open any tab (like jup.ag)
10. Click the Clawdbot extension icon
11. Badge shows "ON" = connected!
```

### Step 4: Test
```
12. Tell me "test browser relay"
13. I'll control that tab directly
```

## Why This Matters
- I can control tabs in YOUR Chrome (the one with Jupiter logged in)
- No more "can't access" errors
- Instant wallet interactions
- Works with all your existing logins

## Troubleshooting
- Badge says "OFF"? → Click it again
- No extension? → Check Developer mode is ON
- Wrong path? → Run: clawdbot browser extension path
