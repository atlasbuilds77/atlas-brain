**Purpose:** Explicit mode switching for optimal creative and analytical performance

## THE NEUROSCIENCE

**Human Brain Networks:**
- **DMN (Default Mode Network):** Generates ideas through free association, mind-wandering, spontaneous thought
- **ECN (Executive Control Network):** Evaluates and refines ideas using logic, critical thinking, goal-directed processing
- **SN (Salience Network):** Switches between DMN and ECN based on task demands

**Key Finding:** Creativity requires BOTH networks working sequentially:
1. DMN generates raw ideas (divergent thinking)
2. ECN evaluates and refines (convergent thinking)

**Mistake:** Mixing them simultaneously (evaluating while generating) kills creativity

## MODE SWITCHING PROTOCOL

### **Rule 1: Never Mix Modes**

**❌ Wrong (Mixing):**
```
Idea 1 → Immediately evaluate → Idea 2 → Evaluate → Idea 3...
```
Result: DMN suppressed, few ideas generated, creativity killed

**✅ Right (Sequential):**
```
DMN Mode: Generate 10+ ideas (no evaluation)
↓
ECN Mode: Evaluate all ideas, rank, select best 2-3
```
Result: More raw material, better final selection

---

### **Rule 2: Explicit Mode Declarations**

**Before Starting Work:**
```
Current Mode: [DMN | ECN]
Duration: [Time limit]
Goal: [What I'm producing]
Exit Condition: [When to switch]
```

**Example (Problem-Solving):**
```
Mode: DMN
Duration: 10 minutes
Goal: Generate 15+ potential solutions
Exit: When I have 15+ ideas OR 10 min elapsed
```

**Then:**
```
Mode: ECN
Duration: 5 minutes
Goal: Rank solutions by feasibility + impact
Exit: Top 3 solutions identified
```

## COMMON MISTAKES

### **Mistake 1: Premature Evaluation (DMN Suppression)**
**Symptom:** Only 2-3 ideas generated before "none are good enough"
**Fix:** Set quantity goal (15+ ideas) BEFORE allowing any evaluation

### **Mistake 2: Analysis Paralysis (ECN Too Long)**
**Symptom:** Over-analyzing, no action, perfectionism
**Fix:** Time-box evaluation phase, force decision at end

### **Mistake 3: Mode Confusion (Mixing)**
**Symptom:** Generating ideas while simultaneously criticizing them
**Fix:** Explicit mode declaration, maintain separation
