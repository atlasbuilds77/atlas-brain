# Order Block Trading Rule - CRITICAL

**Date:** 2026-01-28
**Lesson:** Don't enter positions against order blocks - wait for confirmation

---

## The Rule (From Orion)

**BULLISH ORDER BLOCK (Green Zone):**
- Acts as SUPPORT on retest
- Institutional demand zone
- **DO NOT enter bearish UNTIL it breaks**
- Wait for price to close BELOW the order block before entering puts

**BEARISH ORDER BLOCK (Red Zone):**
- Acts as RESISTANCE on retest
- Institutional supply zone
- **DO NOT enter bullish UNTIL it breaks**
- Wait for price to close ABOVE the order block before entering calls

---

## Why This Matters

Order blocks are areas where:
- Large institutional buying/selling occurred
- Price memory exists
- Reversals are highly likely
- Momentum can stop dead

**Trading AGAINST order blocks = low probability**

---

## Correct Entry Process

### For Bearish Trades (Puts)
1. Identify support levels (bullish order blocks)
2. Wait for price to approach
3. **ONLY enter if price BREAKS AND CLOSES below**
4. Confirmation = volume + follow-through
5. If it bounces = thesis invalidated

### For Bullish Trades (Calls)
1. Identify resistance levels (bearish order blocks)
2. Wait for price to approach
3. **ONLY enter if price BREAKS AND CLOSES above**
4. Confirmation = volume + follow-through
5. If it rejects = thesis invalidated
