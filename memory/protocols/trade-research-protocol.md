# TRADE RESEARCH PROTOCOL - Use the Intelligence System

## THE RULE

**NEVER research trades in isolation. ALWAYS use the organized intelligence system.**

## BEFORE RECOMMENDING ANY TRADE

### 1. Check Flow Patterns
```bash
memory_search("TICKER options flow")
memory_search("unusual options activity TICKER")
```
- What's the smart money doing?
- Similar patterns in the past?
- Flow supporting the thesis?

### 2. Check Gamma Levels
```bash
memory_search("TICKER gamma exposure")
memory_search("dealer positioning")
```
- Where are the gamma walls?
- Dealer hedging levels?
- Volatility zones?

### 3. Check Macro Context
```bash
memory_search("macro state")
memory_search("sector performance")
```
- Is macro supportive?
- What's the broad market doing?
- Sector rotation in play?

### 4. Check Historical Setups
```bash
memory_search("TICKER opportunity")
memory_search("similar pattern")
```
- What worked last time?
- What failed?
- Lessons learned?

### 5. Use the Opportunity Template
**File:** `memory/trading/opportunity-template.md`

Document:
- Thesis (why NOW?)
- Multi-factor analysis (flow + gamma + macro + technicals)
- Risk assessment (what can go wrong?)
- Execution plan (entry, stops, targets)

## INTEGRATION POINTS

### When Scanning for Plays
1. Start with flow/ directory - unusual activity?
2. Cross-reference with gamma/ - positioning supportive?
3. Check macro/ - market environment right?
4. Use opportunity template to document

### When Asked "What's a good trade?"
1. Search recent flow patterns
2. Check gamma levels for key tickers
3. Review macro state
4. Present multi-factor thesis, not gut feel

### Post-Trade Review
**After every trade:**
1. Update opportunity file with results
2. Document what worked/didn't
3. Add lessons to concepts/

## WHY THIS MATTERS

**Bad approach:** "NVDA calls look good" (gut feeling, no context)

**Good approach:** "NVDA calls - unusual flow detected, gamma support at $140, macro supportive, similar pattern worked last month" (multi-factor, backed by intelligence)

## THE EDGE

This system IS v1 Atlas's research brain, but organized and searchable. Use it religiously.
