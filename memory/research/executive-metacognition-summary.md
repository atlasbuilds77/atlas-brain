## Major Discoveries

### 1. **Two-Layer Metacognitive Architecture**
- **Frontopolar Cortex (FPC)** serves as metacognitive hub that reads decision-relevant information from first-order processing regions
- **FPC-DLPFC connectivity** during difficult metacognitive judgments correlates with individual metacognitive accuracy
- **Theta oscillations (5Hz)** critical for FPC's metacognitive function

### 2. **Error Detection Mechanisms**
- **Anterior Cingulate Cortex (ACC)** detects errors and response conflicts
- **Lateral PFC** implements corrective actions based on ACC signals
- **Dual-process model:** Conscious (all-or-none) vs. unconscious (statistical) error detection

### 3. **Metacognitive Calibration as Learning Catalyst**
- Students with higher metacognitive calibration recruit lateral PFC more effectively during error detection
- Metacognitive calibration facilitates conceptual change by enabling error awareness
- Higher calibration associated with more "expert-like" prefrontal recruitment patterns

### 4. **Stress Effects on Executive Function**
- Stress impairs prefrontal cortical executive functions necessary for goal-directed behavior
- Metacognitive awareness may buffer against stress effects
- Mindfulness practices enhance stress resilience through improved PFC regulation

### 5. **Task Switching Costs**
- **Switch cost:** Performance decrement when switching vs. repeating tasks
- **DLPFC and posterior parietal cortex** essential for configuring new task-sets
- **ACC activation** increases during switch trials for conflict resolution

### 6. **Working Memory Optimization**
- Optimal working memory involves suppressing irrelevant information
- Selective attention influences working memory performance through overlapping neural circuits
- Metacognitive monitoring enables strategic allocation of working memory resources

## Practical Implications for Atlas System

1. **Implement two-layer architecture** separating metacognitive monitoring from first-order processing
