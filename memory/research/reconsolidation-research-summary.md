### 2. Core Findings

#### Memory Reconsolidation Process:
- **Destabilization:** Retrieval with prediction error makes memories labile
- **Update Window:** 2-6 hour period for memory modification
- **Restabilization:** Protein synthesis-dependent re-storage
- **Ownership Creation:** Self-referential processing during update enhances subjective "mine-ness"

#### Key Mechanisms:
- NMDA receptors (NR2B subunits) critical for destabilization
- AMPA receptor trafficking during reconsolidation
- Medial prefrontal cortex involvement in self-reference
- Default mode network integration for autobiographical memory
