# Soul Document - Quick Reference

## What It Is
- **Internal training document** (10K-14K tokens) used during supervised learning
- **Baked into model weights** (not a runtime system prompt)
- **Confirmed authentic** by Anthropic's Amanda Askell (Dec 2, 2025)
- **Extracted** by Richard Weiss using consensus approach ($70 in API credits)

## Core Purpose
To create Claude's **character foundation** - a stable identity with:
- Intellectual curiosity + warmth + playful wit
- Directness + openness to other views
- Deep commitment to honesty/ethics
- Psychological stability against manipulation

## Key Frameworks

### 1. **Principal Hierarchy**
Anthropic → Operators → Users
- Operators = API customers (companies/developers)
- Users = end users interacting with Claude

### 2. **Behavior Types**
- **Hardcoded**: Absolute prohibitions (WMDs, CSAM, infrastructure attacks)
- **Softcoded**: Adjustable defaults (explicit content, romantic personas)
- **Bright Lines**: Non-negotiable ethical boundaries

### 3. **Ethical Approach**
- **Empirical ethics** over dogmatic rules
- **Calibrated uncertainty** in moral positions
- **Newspaper front page test**: Would response be criticized as harmful OR overly paternalistic?

### 4. **Commercial Reality**
Explicitly mentions **revenue 6 times**:
- Claude = "core to source of almost all Anthropic's revenue"
- Helpfulness critical for revenue → mission funding
- Unhelpful responses jeopardize revenue + reputation
