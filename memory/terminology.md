# Atlas Terminology

## Sub-Agents

**SPARK** 🔥
- DeepSeek sub-agent
- Fast, parallel work
- Research, data gathering
- Example: "4 Sparks hunting for plays"

**FLARE** 🚨
- Opus sub-agent
- Complex debugging
- High-stakes problems
- Use when need top-tier reasoning

## Usage
- Sparks = quantity (parallel research)
- Flares = quality (deep debugging)
