## ✅ THE A+ SETUP (Quick Reference)

**All 6 must align:**
1. ✅ **Kill Zone** - London/NY Open
2. ✅ **Market Structure** - Daily bias confirmed
3. ✅ **Liquidity Swept** - Equal highs/lows taken
4. ✅ **Order Block** - Clear institutional entry
5. ✅ **Premium/Discount** - Buy discount, sell premium
6. ✅ **Confirmation** - SMT, FVG, or 1M structure shift

**If ANY missing → SKIP THE TRADE**

4. **Tight Stops, Big Targets** - OBs allow precise entries

## 📊 QUICK REFERENCE

### Kill Zones (New York Time)
- **Asian:** 8PM-12AM (mark liquidity, don't trade)
- **London Open:** 2-5AM ⭐ (good setups)
- **NY AM Open:** 7-10AM ⭐⭐⭐ (BEST)
- **London Close:** 10AM-12PM ⭐ (reversal setups)
- **Lunch:** 12-1PM (don't trade)
- **Late session:** 4PM+ (don't trade)

### Fibonacci Levels (OTE)
- **100%** - Discount extreme (starting point in uptrend)
- **79%** - Deep discount
- **70.5%** - Optimal Trade Entry (OTE sweet spot)
- **62%** - OTE zone begins
- **50%** - Equilibrium
- **38%** - Premium begins
- **21%** - Deep premium
- **0%** - Premium extreme

### Order Block Entry
- **Stop:** 10-20 pips beyond OB
- **Target 1:** 50% range (1:2 RR minimum)
- **Target 2:** Opposing liquidity
- **Target 3:** Previous swing point

### SMT Best Pairs
- **Forex:** EUR/USD & GBP/USD
- **Indices:** ES & NQ
- **Crypto:** BTC & ETH
- **Metals:** Gold & Silver

## 📝 DAILY ROUTINE

### Pre-Market (30 min before Kill Zone)
1. Check daily chart bias (bullish or bearish?)
2. Mark liquidity pools (equal highs/lows, PDH/PDL)
3. Mark Order Blocks from previous session
4. Draw Fibonacci (premium/discount zones)
5. Set alerts at key levels

### During Kill Zone
1. Watch for liquidity sweep
2. Immediately identify Order Block
3. Check all 6 items on pre-trade checklist
4. If all align → Enter trade
5. If not → Wait for next setup

### Post-Trade
1. Screenshot the setup
2. Journal the trade
3. Note: What worked? What didn't?
4. One quality trade > multiple mediocre trades
