# Pre-Trade Entry Checklist

**USE THIS BEFORE EVERY TRADE**

## 1. Direction & Thesis
- [ ] Clear directional bias (bullish/bearish/neutral)
- [ ] Specific catalyst or setup identified
- [ ] Conviction level (0-10): _____

## 2. Key Levels Identified
- [ ] Support levels marked
- [ ] Resistance levels marked
- [ ] Previous highs/lows noted

## 3. ⚠️ ORDER BLOCKS (CRITICAL)
- [ ] **Bullish order blocks identified** (green zones = support)
- [ ] **Bearish order blocks identified** (red zones = resistance)
- [ ] **Confirm NO order block in immediate path**
- [ ] If order block present: **WAIT FOR BREAK before entering**

**Rule:** Never enter bearish near bullish order block. Never enter bullish near bearish order block. Wait for CONFIRMED BREAK with volume.

## 4. Entry Confirmation
- [ ] Price action confirms direction (breakout/breakdown)
- [ ] Volume supports move
- [ ] Not entering against order block
- [ ] Entry level makes sense (not chasing)

## 5. Risk Management
- [ ] Position size calculated (% of capital)
- [ ] Stop loss level defined
- [ ] Profit target defined
- [ ] Risk/reward ratio acceptable (min 1:1.5)

## 6. Timing
- [ ] Time until expiration (0DTE/1DTE/2DTE+)
- [ ] Major catalysts ahead? (FOMC, earnings, etc.)
- [ ] Exit deadline defined (especially for 0DTE)
- [ ] Market session (opening, midday, power hour)

## 7. Final Check
- [ ] All above confirmed
- [ ] Gut check - does this feel right?
- [ ] Capital available
- [ ] Ready to execute

---

**If ANY checklist item fails = DON'T ENTER**

Better to miss a trade than force a bad one.
