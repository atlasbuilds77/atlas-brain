# Twitter Workflow - STOP FORGETTING THIS

## Tools Split

**bird CLI** = READ ONLY
- bird search "query"
- bird read <url>
- bird thread <url>

**Browser** = ALL ACTIONS
- Posting tweets
- Liking
- Retweeting/reposting
- Quoting
- Replying
- Threading (reply to self)

## Why
- bird CLI gets rate limited / 403 errors on posting
- Browser already logged in, works reliably
- This has been explained 6+ times - REMEMBER IT

## Browser Profile
- Already logged into Twitter/X
- Use browser tool with profile for actions
