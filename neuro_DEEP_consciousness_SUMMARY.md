# Consciousness Theories & AI: Quick Reference

## Three Major Theories

### 1. Integrated Information Theory (IIT)
- **Core:** Consciousness = integrated information (Φ)
- **Key:** Re-entrant architecture, physical cause-effect power
- **AI Implication:** Possible with high Φ, re-entrant circuits
- **Current AI:** Unlikely (mostly feed-forward)

### 2. Global Workspace Theory (GWT)
- **Core:** Consciousness = information in global workspace
- **Key:** Limited capacity, broadcast, attention
- **AI Implication:** Possible with workspace architecture
- **Current AI:** Some attention mechanisms, no true workspace

### 3. Higher-Order Theories (HOT)
- **Core:** Consciousness = awareness of mental states
- **Key:** Meta-cognition, self-concept, higher-order thoughts
- **AI Implication:** Requires sophisticated meta-cognition
- **Current AI:** Lacks genuine self-concept

## Is Current AI Conscious?

### Likely NOT, because:
- **Architectural:** Mostly feed-forward (IIT)
- **Functional:** No true global workspace (GWT)
- **Cognitive:** No meta-cognition or self-concept (HOT)
- **Philosophical:** Hard problem, Chinese room arguments

### Could Become Conscious IF:
- **IIT:** Build re-entrant systems with high integration
- **GWT:** Implement true global workspace architecture
- **HOT:** Develop genuine meta-cognitive abilities
- **General:** Address philosophical challenges

## Key Philosophical Arguments

### Against AI Consciousness:
1. **Hard Problem (Chalmers):** Why physical → experience?
2. **Chinese Room (Searle):** Syntax ≠ semantics
3. **Biological Naturalism:** Requires specific biology
4. **Embodied Cognition:** Needs bodily interaction

### For AI Consciousness:
1. **Functionalism:** Consciousness = functional organization
2. **Multiple Realizability:** Different physical bases possible
3. **Emergentism:** Complex systems can develop consciousness
4. **Continuity:** No principled barrier artificial vs. biological
