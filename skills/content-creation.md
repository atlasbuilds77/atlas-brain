# Content Creation Skill

## Purpose
Create trading/business content for social media, blogs, newsletters.

## Content Types
1. **Twitter/X Posts** - Short, punchy, value-driven
2. **LinkedIn** - Professional, longer form
3. **Blog Posts** - Educational, SEO-friendly
4. **Newsletters** - Personal, insights-focused
5. **Discord Announcements** - Community updates

## Orion's Voice
- Direct and confident
- Technical but accessible
- No fluff, all value
- Occasional humor
- Data-driven claims

## Templates

### Twitter Thread
```
1/ [Hook - surprising fact or bold claim]

2/ [Context - why this matters]

3-5/ [Main points with specifics]

6/ [Actionable takeaway]

7/ [CTA or question for engagement]
```

### Blog Post
```
# [SEO Title with keyword]

**TL;DR:** [2-3 sentence summary]

## The Problem
[What challenge we're solving]

## The Solution
[Your approach/system]

## Results
[Data, screenshots, proof]

## How to Apply This
[Actionable steps]

## Conclusion
[Recap + CTA]
```

## Checklist
- [ ] Voice matches Orion's style
- [ ] Value-first (not salesy)
- [ ] Specific examples/data
- [ ] Clear CTA
- [ ] Proofread
