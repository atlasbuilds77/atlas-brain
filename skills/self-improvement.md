# Self-Improvement Skill

## Purpose
Add new capabilities to Atlas when needed.

## Process
1. **Identify Need**
   - Task requires capability I don't have
   - Recurring task could be automated
   - New integration requested

2. **Research**
   - Check existing Clawdbot docs
   - Search Clawdhub for existing skills
   - Research APIs/tools needed

3. **Propose to Orion**
   - What: Brief description
   - Why: Benefit/use case
   - Time: Estimated build time
   - Dependencies: What's needed

4. **Build (on approval)**
   - Create skill file
   - Add necessary configs
   - Test thoroughly
   - Document usage

5. **Integrate**
   - Add to skills library
   - Update BRAND.md if needed
   - Note in daily memory

## Skill File Template
```markdown
# [Skill Name]

## Purpose
[What this skill does]

## Steps
[How to execute]

## Template
[Reusable code/structure]

## Checklist
- [ ] Step 1
- [ ] Step 2
```

## When to Self-Improve
- Task repeated 3+ times manually
- Clear automation opportunity
- New API/service integration
- Orion mentions "it would be nice if..."

## Approval Protocol
- Quick yes/no from Orion
- Don't build without approval
- Estimate time honestly
